# Clean Landing Page (HTML & CSS)

## Project Overview
This project is a simple and clean landing page built using only HTML and CSS.  
It focuses on practicing layout design, spacing, alignment, and building visually balanced web pages.

---

## Purpose of This Project
- Improve HTML and CSS fundamentals  
- Practice real-world landing page layouts  
- Strengthen Flexbox and positioning skills  
- Learn spacing, structure, and visual hierarchy  

---

## Tech Stack
- HTML5  
- CSS3  

---

## Features
- Simple navigation bar layout  
- Clean hero section design  
- Minimal and modern UI style  
- Basic responsive structure (if implemented)  
- Hover effects for buttons and links  

---

## What I Learned
- Structuring web pages properly  
- Using Flexbox for alignment  
- Creating visually balanced layouts  
- Improving spacing and typography  
- Building simple but attractive UI designs
